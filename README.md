# cricket-ai-analysis
Cricket AI Shot Analysis

This project analyzes user cricket batting videos and compares them with famous player shots.

Workflow:

1 User uploads batting video  
2 System extracts frames  
3 Pose detection using MediaPipe  
4 Pose compared with dataset of famous players  
5 Closest match detected  
6 Comparison video generated  

Tech Stack

Python  
FastAPI  
OpenCV  
MediaPipe
